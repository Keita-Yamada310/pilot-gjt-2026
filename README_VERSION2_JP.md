# Pilot GJT Version 2

## 主な改善点

- 1〜6のボタンを大きくし、等間隔で表示
- 両端に「明らかに非文法的」「明らかに文法的」を表示
- 本課題30項目だけを反映する進捗バー
- 項目ごとの提示順を `presentation_order` として記録
- 文の語数を `sentence_word_count` として記録
- 保存画面が消えず、保存成功・失敗を明確に表示
- 保存成功時：OSFだけに保存し、参加者PCにはCSVを作らない
- 保存失敗時：CSVを参加者PCへバックアップ
- 終了画面に参加者番号、完了項目数、時間切れ数を表示

## GitHubでの更新方法

1. ZIPを展開する。
2. repositoryの `Code` を開く。
3. `Add file` → `Upload files`。
4. Version 2フォルダ内の全ファイルをアップロードする。
5. 同名ファイルの置換を確認する。
6. `Commit changes`。
7. 1〜3分後、実験ページを `Ctrl + F5` で強制更新する。

## 動作確認

参加者番号を `v2test001` とし、最後まで実施してください。

成功時：
- 終了画面に「保存が完了しました」
- 「保存先：OSF（DataPipe）」
- PCにCSVはダウンロードされない
- OSFに `pilot_gjt_v2_v2test001_....csv` が作成される

失敗時：
- 終了画面に「オンライン保存に失敗しました」
- PCにCSVがバックアップされる
